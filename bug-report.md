# Bug Report

**Judul Bug:** Login Error ketika menggunakan akun valid
**Tingkat Bug:** High
**Langkah Perbaikan:**

1. Buka halaman login.
2. Masukkan email dan password yang valid.
3. Klik tombol "Login".
4. Terjadi error 500.

**Luaran yang Diharapkan:**
Pengguna dapat login dan diarahkan ke dashboard.
